# Velocity: Solana Wallet

A minimal web wallet for generating seed phrases, deriving Solana accounts, and viewing on-chain balances.

## Features

* Seed phrase generation
* Multiple wallet derivation
* SOL balance lookup

## Environment Variables

```bash
HAPI_KEY=your_helius_api_key
```

## Development

```bash
bun install
bun dev
```

## Tech

* TypeScript
* Next.js
* @solana/kit
* bip39
* tweetnacl
* bs58

## Roadmap

* Ethereum wallet support
* SPL token balance lookup (e.g. USDC)
